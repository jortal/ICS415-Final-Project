Features

MySQL Backend

This application contains a MySQL 5.6 backend for persistence. If the server is restarted at any time, the user data should be kept in tact.

User Login

This application supports user logins. Different users have different levels of power, for example, only the administrator is allowed to create and delete users. Only registered users are allowed to see the 'Current Users' page.

Responsive Web Design

This application makes use Twitter Bootstrap to give web site a responsive design that will adjust the layout based on the window size.
